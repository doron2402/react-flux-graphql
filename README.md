## React, Flux, GraphQL and more Buzz Words

## Requirement
1. I'm using node 5.6.0 but you can use node > 5
2. babel-cli and nodemon globally
3. Mongodb Running on your localbox

## How to start

1. `git clone`
2. `npm i`
3. `npm build` that will compile src/ to dist/
4. `npm start`
5. `webpack -wd`
